Nombre
Entrega Estimada - Smart Modules

Descripción Breve
Permite a tus clientes ver la fecha estimada de la entrega de un producto mientras navega por tu página y mejora tus ventas. Ahora también puedes mostrarlas como Amazon.

Beneficios

Mejora tus ventas y confianza permitiendo a tus clientes poder ver la fecha estimada de entrega para tus pedidos.

Con este módulo Facilitarás al cliente la posibilidad de saber cuándo le llegarán sus productos desde el momento en que entre en la página de producto.

Entrega Estimada de Smart Modules te permitirá configurar múltiples opciones de manera fácil y cómoda como por ejemplo como se mostrará, donde, el formato y los colores para personalizar el modo en que se muestra la información, sin editar una sola línea de código.

Elige los días que preparas los pedidos, la hora límite para enviar un pedido el mismo día, los días que reparten tus transportistas, las entregas… todo desde una única pantalla de administración.

Además también podrás configurar los días festivos para que el sistema omita esos días para las fechas estimadas y el envío de los paquetes.

Todo ello con nuestro destacado servicio al cliente con soporte gratuito para la configuración del módulo.

Características

Elige como, donde y de qué manera se mostrará la caja de información

Fácil instalación

Elige los días de reparto para cada transportista

Elige la entregabilidad para cada transportista
