Web Browser Push Notifications using OneSignal - Module for PrestaShop
==================================================
## Description
Send push notifications to web browsers in any device, even if your shops website is closed, using advanced PrestaShop based segmentation filters.

## Requirements
- [PrestaShop 1.6.x - 1.7.x](https://www.prestashop.com/)
- [Free account in OneSignal](https://onesignal.com/)
- [PHP 5.6+](http://www.php.net/)
- [PHP JSON extension](http://php.net/manual/en/book.json.php)
- [PHP cURL extension](http://php.net/manual/en/book.curl.php)
