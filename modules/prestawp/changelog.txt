# Changelog


## [1.9.1] - 2024-01-17
### Added
- New option "The category page URL"
### Improved
- Smarty cache management
### Fixed
- Missing button "Write a comment" if the post doesn't have comments yet
- Missing loading of masonry.pkgd.min.js due to specific settings configuration
- Disabled the search engine indexation of the product list pages


## [1.9.0] - 2023-11-04
### Added
- Category pages
- Options for loading blocks of posts by Ajax ("Dynamically load the block")
- New option "Maximum number of posts" for product posts
- Notification about outdated WP plugin version
- New option "HTML classes"
- A search form on the post list page
### Improved
- Minor improvements in post selection for products
- Minor settings appearance changes
- Now option "Maximum length of displayed content, characters" in custom blocks also works when showing featured images
- Comment links in the comments block now lead to the PrestaShop post page (when enabled)
- Minor style improvements
- Updated the module code to comply PrestaShop coding standards
### Fixed
- Invalid post preview text when post content starts from the style tag


## [1.8.4] - 2023-08-15

### Fixed
- Incorrect display of comment author name in certain cases
- Selection of WP categories on the product edit page
